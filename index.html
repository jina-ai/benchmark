Benchmark #

To do the resource benchmarking (CPU & Memory usage) of Jina features, you can use scripts/benchmark.sh file. This file actually benchmark scripts under src directory with cmdbench utility tool.

Run #

Run Locally #

bash -x scripts/benchmark.sh

Run on Docker #

JINA_VER=master
docker build --build-arg JINA_VER=$JINA_VER -t bechmark .
docker run -v $(pwd):/app bechmark:latest
